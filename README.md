# randomrestaurant

Returns random restaurants or other locations from Google Maps

```text
usage: randomrestaurant.py [-h] [--version] [--debug] [--config CONFIG] [--radius RADIUS]
                           [-n N] [--keyword KEYWORD]
                           location

positional arguments:
  location              The geographic location to search

options:
  -h, --help            show this help message and exit
  --version             show program's version number and exit
  --debug, -d           Print exception stacktraces
  --config CONFIG, -c CONFIG
                        The path to the configuration file
  --radius RADIUS, -r RADIUS
                        The radius of the search in meters
  -n N                  TThe maximum number of results to return
  --keyword KEYWORD, -k KEYWORD
                        The keyword to search for (use quotes around multiple keywords
```
